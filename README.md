# spring-kotlin-server

# Compile

```
   mvn compile
   mvn package
```

# Run
+ Dev
```
   java -jar spring-kotlin-server-[version].jar --spring.profiles.active=dev 
```
+ Prod
```
   java -jar spring-kotlin-server-[version].jar --spring.profiles.active=dev 
```

# Docker
```
    docker build -t spring-kotlin-server:[version] .
```
