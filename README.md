# eimmo-user-api-proto

# Compile

```
   mvn compile
   mvn package
```

# Run
+ Dev
```
   java -jar eimmo-use-api-[version].jar --spring.profiles.active=dev 
```
+ Prod
```
   java -jar eimmo-use-api-[version].jar --spring.profiles.active=dev 
```

# Docker
```
    docker build -t eimmo-user-api:[version] .
```
